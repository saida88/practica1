Aplicació web sobre gratacels

Aplicació que permet obtenir dades sobre gratacels, els arquitectes que els han construit, l'estil arquitectònic i els materials utilitzats.

Desarrollo de aplicación web bàsica.

A partir del caso que habeis propuesto, y solo en caso de que el profesorado os haya comunicado que es correcto: Desarrollar, usando Django, una aplicación web básica de acceso a datos (siguiendo el modelo de datos propuesto).

La aplicación ha de:

  1. Permitir listar todas las entidades que forman parte del modelo de datos propuesto.
  2. Acceder a los datos individuales de cada instancia de cada una de las entidades.
  3. Obtener estos datos en 3 representaciones diferentes:
    a. HTML (web “legible” por el usuario, usando templates).
    b. JSON
    c. XML
  4. Todas estas operaciones (listados y acceso individual a cada instancia) deben ser rutas de un API RESTful.
  5. Activar la interfaz admin de Django para añadir, modificar y eliminar instáncias.
  6. Añadir autenticación y gestión de usuarios (builtin de Django).
